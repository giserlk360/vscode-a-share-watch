# A股盯盘

在 VS Code 侧边栏查看 A 股行情和持仓信息的轻量插件。它把自选股、持有股、预购股、指数行情、走势小图和明日计划集中到一个侧边栏面板里，适合在编码时快速扫一眼行情，不需要切换到浏览器或行情软件。

本项目基于 [Mrxulovemingming/vscode-stock-monitor](https://github.com/Mrxulovemingming/vscode-stock-monitor) 二次开发，当前版本已重构为侧边栏 A 股盯盘体验。

## 功能特性

- 侧边栏行情看板：在 VS Code 活动栏打开「A股盯盘」，集中查看行情。
- 多列表管理：支持「自选股」「持有股」「预购股」三个 Tab，分别管理关注、已持仓和计划买入的股票。
- 实时行情：展示当前价格、涨跌幅，并根据涨跌使用不同颜色区分。
- 持仓统计：支持买入价、持仓数量、持仓盈亏、持仓涨跌幅、持仓金额、当日盈亏和总市值统计。
- 指数展示：内置上证指数、深证成指、创业板指、沪深 300、科创 50，可在设置中控制是否显示到列表。
- 批量导入导出：自选股支持按股票代码或名称批量导入，并可导出代码、名称、代码加名称或完整 JSON。
- 走势查看：可在股票列表中查看近 5 日或近 10 日走势小图。
- 明日计划：在侧边栏记录次日交易备忘。
- 自动预购筛选：定时检查自选股近期走势，满足连续下跌或近 5 日大幅下跌条件时自动加入预购股。
- 状态栏轮播：可在 VS Code 底部状态栏轮播展示关注股票行情。

## 使用方式

1. 安装并启用插件后，点击活动栏中的「A股盯盘」图标。
2. 在「股票列表」中切换「自选股」「持有股」「预购股」。
3. 点击添加按钮，搜索股票代码或名称，选择结果后保存。
4. 持有股可填写买入价和持仓数量，用于计算盈亏和市值。
5. 点击走势图按钮查看近 5 日或近 10 日走势。
6. 在「明日计划」中记录次日交易备忘。

## 侧边栏功能

股票列表提供三个视图：

- 自选股：适合日常关注和行情轮播，可批量导入、导出。
- 持有股：适合记录实际持仓，支持买入价、股数、盈亏和市值统计。
- 预购股：适合记录计划买入目标，也可由自动预购筛选加入。

常用操作：

- 添加：搜索股票代码或名称后加入当前列表。
- 删除：从当前列表移除股票。
- 编辑：持有股可编辑买入价和持仓数量。
- 排序：按涨跌幅升序、降序或默认顺序切换。
- 走势：查看近 5 日或近 10 日收盘走势。
- 导入导出：批量维护自选股列表。
- 明日计划：直接在侧边栏记录并保存交易备忘。

## 配置说明

在侧边栏「插件设置」中可以配置：

- 数据刷新频率。
- 股票列表显示字段：股票代码、当前价格、涨跌幅、买入价格、持仓数量、持仓盈亏、持仓涨跌幅、持仓金额。
- 指数列表显示：控制上证指数、深证成指、创业板指、沪深 300、科创 50 是否展示在股票列表中。

## 数据来源

行情数据优先从新浪财经接口获取，失败时回退到东方财富接口，并在网络异常时尽量使用内存缓存数据。股票搜索和名称解析使用东方财富搜索接口。

当前主要面向沪深 A 股和常见 A 股指数，暂不覆盖港股、美股和基金等完整市场。

## 开发

```bash
npm install
npm run build
npm run compile
npm run test:unit
```

常用脚本：

- `npm run build`：使用 `build.js` 构建扩展入口。
- `npm run watch`：监听源码并持续构建。
- `npm run compile`：执行 TypeScript 编译检查。
- `npm run test:unit`：运行单元测试。
- `npm run test:property`：运行属性测试。

## 发布准备

插件发布到 VS Code Marketplace 前，建议检查以下内容：

- `package.json` 中的 `publisher`、`repository`、`displayName`、`description` 是否为最终发布信息。
- `resources/icon.png` 是否为 128x128 或更高分辨率 PNG 图标。
- `resources/icon.svg` 是否适合作为 VS Code 活动栏图标。
- 执行 `npm run build` 或 `npm run compile`，确认扩展可以正常构建。
- 如需打包 `.vsix`，可使用 `vsce package`。
