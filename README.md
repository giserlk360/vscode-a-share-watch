# 注释看盘

在代码注释中实时查看 A 股行情。写代码的同时，悄悄看一眼股票。

## 功能特性

**注释内联显示** — 在代码注释中输入股票代码（如 `600036`）、股票名称（如 `招商银行`）、别名或特殊词汇（如 `上证指数`、`创业板指`），自动在注释后方显示实时行情信息。

支持显示的内容（可在设置中勾选）：
- 当前价格
- 涨跌幅
- 涨跌额
- 持仓盈亏
- 当日盈亏

**隐蔽模式** — 开启后，注释中的行情信息颜色与注释文字一致，不仔细看完全看不出来。

**状态栏轮播** — 在 VSCode 底部状态栏轮播显示持仓股票行情，支持自定义轮播间隔和显示内容。

**价格预警** — 设置目标价格或涨跌幅，触发时弹窗提醒或注释行红色闪烁。

## 使用方法

1. 安装插件后，点击左侧活动栏的股票图标
2. 在「股票列表」中点击 ＋ 添加股票，搜索股票代码或名称
3. 填写买入价格、持仓数量（可选）
4. 在任意代码文件的注释中输入股票代码或名称，即可看到实时行情

```javascript
// 今天看看 600036 的表现
// 创业板指 跌了不少
// 持仓 整体情况如何
```

## 设置说明

在侧边栏「插件设置」中可以配置：

- 隐蔽模式开关
- 数据刷新频率
- 状态栏轮播开关、间隔、显示内容
- 注释显示内容（当前价格、涨跌幅、涨跌额、持仓盈亏、当日盈亏）
- 价格预警方式（弹窗、闪烁、两者都用）
- 特殊词汇别名管理

## 支持的注释格式

- `//` JavaScript/TypeScript/Java/C++
- `#` Python/Shell/Ruby
- `--` SQL/Lua
- `<!-- -->` HTML
- `/* */` 块注释

## 数据来源

行情数据来自东方财富 API，备用数据源为新浪财经。仅支持 A 股（沪深两市）。
